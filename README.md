#Superficial White Matter Atlas Repository

##Description

The repository provides tools for:
  Constructing a population-level superficial white matter fiber atlas using high-resolution diffusion MRI data.
  Performing functional and structural connectivity analyses to investigate the role of SWM in brain networks.

##Software requirements

MATLAB 2024B
Python packages (included in environment.yml):
  - numpy
  - torch
  - scipy
  - nibabel
  - nilearn
  - scikit-learn
  - scikit-image
  - matplotlib
  - pandas
  - statsmodels
  - dipy

others as listed in environment.yml
